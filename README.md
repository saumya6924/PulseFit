PulseFit — Smartwatch E-Commerce Website

PulseFit is a front-end e-commerce website built for a fictional smartwatch brand using HTML, CSS, and JavaScript.
The website includes multiple pages such as a home/landing page, product listing, product details, and a checkout flow. 
It also features dedicated marketing pages covering digital marketing, mobile marketing, video marketing, app marketing, and location-based marketing demos. 
Additionally, the site includes a fitness blog with articles on smartwatch health tracking, sleep recovery, and product comparisons.
Developed as part of an academic project for IMCA Sem-4 at LJ University using VS Code.
